# Todo App

Java のバックエンドと React のフロントエンドで構成したフルスタックの Todo アプリです。

基本は Todo を管理する CRUD アプリですが、定期タスク機能を持っているため、単純な Todo アプリより一歩進んだ内容になっています。

定期タスクとして登録した Todo は日課用タスクとして扱われ、cron によって 0 時になると `done` ステータスがリセットされます。これにより、毎日繰り返す習慣やルーティンも管理できます。

そのためこのアプリは、単なる CRUD の練習に加えて、次のような点も学べる構成になっています。

- 複数テーブルの連携
- トランザクション境界の設計
- 定期タスクの管理
- cron を使ったバッチ処理

## 主な機能

- Todo の作成、取得、更新、削除
- 定期タスクの登録、解除
- 毎日 0 時に定期タスクの完了状態を自動でリセット
- 定期タスクの状態をリセットするためのバッチ処理

## 技術構成

- バックエンド: Java
- フロントエンド: React + TypeScript + Vite
- データベース: MySQL
